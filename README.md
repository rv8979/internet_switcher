# internet_switcher
change internet line without using any hardware


Install

cd ~
wget https://github.com/rv8979/internet_switcher/archive/master.zip
unzip master.zip -d ./
cd internet_switcher-master
chmod +x setup.sh
sudo ./setup.sh
